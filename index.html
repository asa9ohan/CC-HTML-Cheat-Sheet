Semantic HTML

What is Semantic HTML?

Semantic HTML is the use of descriptive HTML tags in HTML code in place of generic tags such as <div> and <span> that do not give any clues about their content.

Why use Semantic HTML?

  • Accessibility: screen readers and browsers have more context when interpreting code.
  • SEO: improves site's SEO, allowing search engines to identify content and appropriately weight them.
  • Easy to understand: makes source code easier to read for other developers.

Opening Tag Closing Tag Description
<header> </header> A container usually for either navigational links or introductory content (<h1> to <h6> headings).
<nav> </nav> Used to define a block of navigation links such as menus and tables of contents.

🚩 Can be used inside of <header> element, or on its own.
